Simon Says Game

Press any key to start the game

How to Play:

Starting the Game:

  • Turn on the Simon game.
  • The game will randomly light up one of the four colored buttons

Repeating the Sequence:

  • After the game shows the sequence, your task is to press the buttons in the same order.
  • If you successfully repeat the sequence, the game will add another light to the sequence.

Increasing Difficulty:

  • The game continues to add one new light to the sequence each round.
  • Your task is to remember and repeat the entire sequence, which grows longer each round.

Making a Mistake:

  • If you press the buttons in the wrong order, the game will over

Winning:

  • The game usually continues until the sequence becomes too long to remember, or until you make a mistake. Some versions might have a predefined length to reach to win.
